Characteristics of Networks

What is a computer network?

A computer network is an interconnected system of a number of computers which are connected together in order to share data. Computers connected on a network can do the following:

  • Share peripherals and other hardware between multiple devices such as
    • Printers
    • Storage
    • Routers
    • And any other device that can be connceted to a network
  • Create, Send and Share files over the network. Abillity to save work on other connected devices and access the work on any connected device.
  • Share computational resources between devices

Hardware

Network Hardware

To set up a computer network you need some hardware for it.

  • Network Cable

    Network cables are used to connect wired devices together. The typical consumer grade cable used is an ethernet cat5e cable with an RJ45 connector. The cable is capable of 1000Mbps (gigabit) speeds between devices. A more "prosumer" or business may use Cat6 cable which is up to 10Gbps

  • Router

    A router is a device that is a device that all other devices on a network will connect to. As the name suggests it routes data to the correct location. Modern routers also have a wireless module which allows devices to connect wirelessly. A router is not to be confused with a modem, The modem is what connects a network to the internet.

  • Network Interface Cards (NIC)

    For a device to connect to the internet it requires a network interface card which is also known as a NIC. There are two types of NIC, Wireless and Wired. The wired variety typically takes the aforementioned RJ45 Connection. Most devices will have these built it. Modern computers typically have a wired NIC built into the motherboard more expensive ones may have a wireless one as well. If that is not the case they can be bought seperately by adding it to a PCI/PCI-E Slot on your motherboard.

Topology

Network Topologies

When a network is created devices are not just connected randomly. That are connected in a certain layout, usually one of the many number of recognised layouts.

  • Ring Network

    In the ring network topology all computers are connected in a closed loop or a "ring". Each device is connected to two others, one on its "left" and one on its "right". Every computer is either directly connected or indirectly conneted.

  • Bus Network

    In the bus network topology every computer is directly connect to one "backbone" cable called the bus. All the data is passed up and down this cable. In this network every computer is effectively connected directly together.

  • Token Ring Network

    If the Token Ring protocol is used in a star or ring topology the data will only travel in one direction. The signal is carried by a "token" from node to node.

  • Mesh Network

    In a full mesh network toplogy every computer is connected to ever other directly. In a partial mesh network some computer are connected to all the others and some are only connected to the nodes which they exchange the most data with.

  • Tree Network

    In the tree topolgy is composed of multiple star networks with the central computer connected to the main bus. Therefore, A bree network is a bus network of star networks.
